# Générateur de bâtonnets de semis 🌱

Application web pour générer des fichiers SVG de bâtonnets de semis personnalisés.

## Déploiement

Ce dossier contient la version prête pour le déploiement.

## Utilisation

Ouvrez `index.html` dans un navigateur web.

## Fichiers

- `index.html` - Application principale
- `varietes.json` - Liste des variétés de légumes
- `.nojekyll` - Fichier pour GitHub Pages
